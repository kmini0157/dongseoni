# 동선이 · 전국 여행 비서

가고 싶은 곳·맛집·숙소를 넣으면 **날씨·영업시간·이동시간을 엮어 동선을 짜주고**, "지금 어디서 몇 분 뒤 어디로"를 한 화면으로 보여주는 여행 비서.

서버 비용 0원 · 가입 없음 · 추적/광고 없음. 단일 `index.html` 정적 웹앱(PWA).

라이브: https://kmini0157.github.io/dongseoni/

---

## 화면 구성 (4탭 + 마법사)

1. **여행 만들기 마법사** — 첫 실행·새 여행 시 "어디로 → 언제·며칠 → 이동수단" 3단계로 안내
2. **🏠 오늘** — 지금 어디서, 몇 분 뒤 어디로. 실시간 라이브 뷰
3. **🗓️ 일정** — Day별 동선 타임라인. 카드를 꾹 눌러 끌면 순서 변경, 📌 시각을 고정하면 '최적 코스 자동 정리'가 그 시각을 기준으로 나머지를 배치
4. **➕ 담기** — 장소 수집의 단일 입구. **보관함**(모아두면 자동 코스) / **Day에 바로**(직접 추가) 토글
5. **🧳 여행** — 날짜별 도시·시작시각·숙소 한눈에, 도착·끼니·공유·API·초기화

## 주요 기능

- **단일 화면 동선** — 권역끼리 묶어 도착/출발 시각·이동시간을 계산, 지금/다음만 압축해서 표시
- **최적 코스 자동 정리** — 📌 고정한 시각과 끼니(아침·점심·저녁)는 시간축 앵커로 지키고, 나머지를 **실측 도로거리 행렬**(OSRM table, 1회 호출로 N×N) 기준 최소 동선으로 배치(TSP + 시간창). 강·산·다리를 반영해 직선거리보다 정확
- **🗺️ 나의 여행 발자국 + 완주 세리머니** — 모든 여행의 도착 체크인을 합산해 전국 지도에 정복 권역을 색칠(9개 권역·도시·방문 수 카운터). Day별 진행률(2/5 곳 방문)을 Day 탭에 표시하고, 한 Day의 모든 곳을 밟으면 🎉 컨페티+완주 배너, 계획한 곳을 다 방문하면 🏆 여행 완주 트로피 + 발자국에 🏅 완주 스탬프
- **🔗 공유 루프 (공동 계획)** — 링크로 받은 일정을 별도 여행으로 "추가"하거나, 장소만 지금 여행 **보관함에 합치기**로 담아 함께 코스를 짬(서버 없이, 이름 중복 자동 정리·되돌리기 지원)
- **멀티데이·멀티시티** — 며칠 일정 + 날짜별 다른 도시·숙소
- **전국 지원** — 지도에서 도(道)→도시 클릭 선택, 전국 162개 시/군 자동 등록
- **실시간 날씨 + Plan B** — Open-Meteo(무키) 또는 Visual Crossing(키 등록 시 — 상업 허용). 비 확률 60%↑ 또는 미세먼지 나쁨 시 **실내 대안(Plan B)** 제안 — 카카오 카테고리 검색으로 지금 위치 반경의 문화시설·대형마트·카페를 거리순 추천(전국, 하드코딩 폴백). 비 오면 시간별 강수 스트립으로 "몇 시쯤 갬" 안내, 타임라인에 🌅 일출·🌇 일몰 라인 표시 + 노을 골든아워 배지
- **카카오맵으로 코스 보기** — 하루 동선을 출발+경유(최대 5곳)+도착 길찾기 링크로 한 번에. 시작점을 직접 골라 거기서부터 이어볼 수 있고, 7곳 초과 시 구간 자동 분할(WCONGNAMUL 좌표 변환 내장)
- **구간 이동시간 직접 확인** — 타임라인 이동 구간을 탭하면 카카오맵(실시간 교통)에서 소요시간을 확인하고 분 단위로 직접 입력. 자동 계산보다 우선 적용되고 공유 링크로 동행자에게도 전달
- **🩺 여행 건강검진** — 여행 탭에서 전 일정을 한 번에 점검: 휴무·라스트오더·시간 역행·빠듯한 이동·빈 끼니·강수 60%↑·막차 마지노선(로컬)에 더해, **공휴일**(천문연 특일정보)·**미세먼지**(에어코리아)·**관광지 혼잡**(관광공사 집중률)까지 확인. 새 3종은 모두 data.go.kr이라 지금 쓰는 관광공사 키로 동작(해당 서비스만 무료 활용신청)
- **당일 어긋남 원터치 복구** — 체크인이 계획과 ±15분 이상 어긋나면 "지금 기준 재계산" 제안(이후 일정 자동 전파), 휴무·만석 장소는 "건너뛰기"로 보관함에 옮기고 즉시 재계산
- **맛집·놀거리 추천** — 카카오 검색 인기·추천순 정렬 + 한국관광공사(TourAPI) 공식 명소·사진. 실제 평점은 카카오맵 딥링크로 위임(약관 안전)
- **🍜 뭐 먹지? (후기 종합)** — 맛집마다: ①TourAPI 공식 대표메뉴·영업시간 ②카카오맵 공개 후기 본문 + 모바일 별점 목록(~20개) 이중 수집 → 별점 평균·분포·태그(맛·가성비)로 **후기 기조**(😋 호평 위주/🤔 호불호/⚠️ 아쉬움 많음) 사실 판정 ③음식 사전 기반 인기 메뉴 랭킹 ④Gemini 키가 있으면 **총평 한 줄·메뉴·팁(웨이팅·주차)·아쉬운 점**까지 정리. 후기 원문은 세션 메모리만(저장·공유 안 함), 참고용
- **블로그 코스 가져오기** — 여행 블로그 본문을 붙여넣으면 장소를 뽑아 일정화
- **이동수단별 시간** — 대중교통(TMAP 키 등록 시 TMAP, 아니면 ODsay 실측)·도보·자동차(Mapbox 키 등록 시 Mapbox 교통 반영, 아니면 OSRM 도로거리 + 도심 속도 보정). 일정이 바뀌면 전 구간을 미리 실측하고, 실측값은 기기에 저장(도로 7일·대중교통 24시간)돼 재시작·오프라인에도 유지. 대중교통은 같은 응답에서 **구간 요금·환승 횟수**도 뽑아 타임라인 배지 + Day 교통비 합산으로 표시(추가 호출 0건)
- **자동 저장·공유·백업** — URL 해시 + localStorage. 링크 하나로 동행자와 공유·복원, 설정에서 JSON 전체 백업 내려받기/불러오기(기기 변경 대비)
- **PWA 마감** — 뒤로가기는 시트만 닫음(앱 종료 안 됨), 자정이 지나면 오늘 Day로 자동 전환, 오프라인이면 "📵 저장된 정보" 모드(스테일 예보 라벨·무한 로딩 방지)

## 외부 데이터 (모두 무료, 키는 기기에만 저장)

| 용도 | 기본 서비스 | 키 | 상업 안전 대체(설정에서 키 등록 시 자동 전환) |
|---|---|---|---|
| 날씨 | Open-Meteo | 불필요 | **Visual Crossing** — 무료 일 1,000레코드·상업 허용·15일 예보(카드 불필요). Open-Meteo 무료판은 비상업 전용 |
| 지도·장소·맛집 검색·자동완성 | Kakao Maps JS | JS 키(도메인 잠금) | (그대로 사용 — 상업 허용, 쿼터 정책만 주의) |
| 대중교통 시간 | ODsay | 웹 키 | **TMAP 대중교통(SK open API)** — 사용자격 제한 없음·종량제 상업 경로. ODsay 무료는 개인·비상업·6개월 한정 |
| 놀거리·관광 명소(사진) | 한국관광공사 TourAPI(KorService2) | 내장(기본 키) | (그대로 사용 — 상업 허용, 출시 전 운영계정 전환 권장) |
| 도보·자동차 경로/거리·최적화 행렬 | OSRM 공개 데모 | 불필요 | **Mapbox Directions/Matrix** — URL 제한 public 토큰, 월 무료 한도 내 상업 가능. OSRM 데모는 무보장·유료화 금지 |

기본 키가 내장돼 있어 바로 동작하며, 본인 키로 바꾸려면 앱 안 설정에서 입력하면 됩니다(기기에만 저장). 키가 전부 막혀도 큐레이션/추정 폴백으로 동작합니다. TMAP·Visual Crossing·Mapbox 키를 등록하면 해당 기능이 상업 이용 안전 소스로 전환됩니다 — 날씨는 호출 실패 시 즉시, 대중교통·도로는 키 무효 판정 시 기존 소스로 자동 폴백합니다.

## 배포 (GitHub Pages, 0원)

1. GitHub repo `kmini0157/dongseoni`에 파일 업로드(덮어쓰기)
2. Settings → Pages → `main` / `(root)` → Save
3. `https://kmini0157.github.io/dongseoni/` 에서 1~2분 뒤 반영

코드 수정 후에는 변경된 파일만 재업로드하면 됩니다.

## 파일 구성

```
index.html              앱 본체(단일 파일)
manifest.json           PWA 매니페스트
sw.js                   서비스워커(network-first, 오프라인 캐시)
privacy.html            개인정보처리방침
icon*.png / icon*.svg   앱 아이콘(192/512/maskable/1024/180/favicon)
feature-graphic.png     Play 피처 그래픽(1024×500)
.well-known/assetlinks.json   안드로이드 TWA 도메인 인증
codemagic.yaml          iOS 클라우드 빌드 설정
_build/                 아이콘·그래픽 생성 스크립트(배포 제외)
```

> `_build/`(node_modules 포함)는 빌드 도구라 GitHub에 올릴 필요 없습니다.
